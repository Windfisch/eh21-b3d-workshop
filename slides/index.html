# Installation Erstmal ein *venv* erstellen, damit `pip install` nicht unser System beschmutzt. ```sh cd path/to/ python -m venv b3d-venv ``` In jedem Terminal: *venv aktivieren* ```sh source path/to/b3d-venv/bin/activate ```
*Visual Studio Code* installieren und starten: ```sh code --enable-proposed-api ms-toolsai.jupyter \ --enable-proposed-api ms-python ``` "File" / "Open folder" und euer Workshop-Verzeichnis wählen.
Python, Python Debugger und ms-toolsai/Jupyter installieren: ![](img/vscode.png)
https://github.com/bernhard-42/vscode-ocp-cad-viewer/releases/download/v2.1.1/ocp-cad-viewer-2.1.1.vsix installieren. ![](img/vscode2.png) Dann ![](img/ocp.png) und "Quickstart build123d".
Und jetzt die gepatchte build123d-Version ```sh source path/to/b3d-venv/bin/activate # nicht vergessen :) pip install \ git+https://github.com/Windfisch/build123d@eh21-workshop ```
# Ran ans Gerät!
### Hello-World ```py # %% from build123d import * from ocp_vscode import * # %% box = Box(10,15,20) show(box) ```
### Transformationen / Location ```py # %% from build123d import * from ocp_vscode import * # %% box = Location((10,0,20)) * Box(10,15,20) box2 = Location((0,0,0), (45, 45, 0)) * Box(10, 10, 10) show(box, box2) ```
### Primitiven ```py # %% from build123d import * from ocp_vscode import * # %% sph = Location((10,0,20)) * Sphere(radius = 10) cyl = Location((0,0,0), (45, 45, 0)) * Cylinder(radius = 5, height = 10) show(sph, cyl) ```
- select top face - create plane from there - make circle on there - extrude - add together - use debugger
